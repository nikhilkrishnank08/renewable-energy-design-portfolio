# Sample Design Package

Reference drawings demonstrating typical deliverables for domestic
renewable energy retrofits under the SEAI One Stop Shop scheme.

## Drawings

### Heat Pump Plant Room Schematic (ARE-M-001)

- [heat-pump-plant-room.pdf](./heat-pump-plant-room.pdf) — viewable PDF
- `heat-pump-plant-room.dxf` — editable source (AutoCAD / LibreCAD compatible)

**Project context**

Typical 3-bed semi-detached domestic retrofit under the SEAI One Stop
Shop scheme. The 8 kW heat pump size is consistent with the output
of the accompanying [heat loss calculator](../heat-loss-calculator/)
for a ~110 m² dwelling.

**Key equipment shown**

- 8 kW air-to-water heat pump (outdoor unit)
- 50 L buffer tank for minimum system volume and defrost cycles
- 200 L DHW cylinder
- Primary circulation pump P1 (heat pump to buffer)
- Secondary circulation pump P2 (buffer to UFH manifold)
- Underfloor heating manifold for ground floor distribution

**Drawing conventions**

- Layers separated by function: equipment, flow pipework, return
  pipework, valves, text, dimensions
- Flow pipework in red, return in blue (standard mechanical convention)
- Title block bottom-right with drawing number and revision
- Drawn in millimetres, scale NTS

## Tooling

Drawings produced in LibreCAD, exported in DXF 2007 format (fully
AutoCAD-compatible) and PDF.
