# Sample Design Package

This folder contains reference design drawings and schematics demonstrating
typical deliverables for a domestic retrofit package under the SEAI One
Stop Shop scheme.

## Intended contents

- Underfloor heating layout (ground floor, typical 3-bed semi)
- Heat pump plant room schematic (air-to-water, buffer tank, cylinder)
- Single-line electrical schematic (heat pump + solar PV + battery + EV point)
- Solar PV roof layout (panel arrangement, string configuration)
